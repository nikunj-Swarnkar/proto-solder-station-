#a soldering station 
this is a whole set up to get started with soldering anywhere !

---

### this set up concists of:-
- a fixed voltagge supply with 12 v , 5 v and 3.7 v 
- a solder holder with a tip cleaner and place for sponge 
- a fume hud with filters 
- a twezeers holder 
- help hands 
- wireless solder 

---

### GOING INTO THIS REQUIRES TO USE 3S CONFIG OF 3.7V BATTERY SO IF U DONT KNOW WHAT YOU ARE DOING , U CAN HURT YOURSELF
with that out of the way lets start
